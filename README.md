# style – D0 · iki1uc

## Zweck
style ist die Form‑Ebene (D0) des 6E‑Würfelsystems.  
Sie definiert Erscheinung, Struktur, Oberfläche und die visuelle Logik aller Module.  
D0 ist die äußere Schicht des Systems und trägt die 12e‑Stabilität in jede Richtung.

style bestimmt:

- Form  
- Rahmen  
- Layout  
- Übergänge  
- Sichtbarkeit  
- Oberflächen‑Verhalten  

D0 ist die Ebene, die entscheidet, **wie** das System aussieht und **wie** es sich zeigt.
