# style – D0 · iki1uc

## Zweck
style ist die Form-Ebene (D0) des 6E-Würfelsystems.
Es definiert die äußere Erscheinung, die visuelle Struktur und die
Oberflächenlogik aller Module und Übergänge.

## Ebenen
- D0 = style  
- D1 = head  
- D2 = HTML  
- D3 = body  
- D4 = ANKER (Master)  
- D5 = API (Slave)

## TMP-Läufer
SYQ verbindet D5 mit META und stabilisiert Übergänge.

## Parität
D4 ↔ D5 bilden die Master/Slave-Achse.

## Regeln
- Exakt 6 Ebenen (Würfel-Gesetz)
- D0 ist die Form-Ebene
- META ist Beobachter, nicht Ebene
